# Profile Images

This folder contains profile images for the "RAIR" app.

## Image Naming Convention

Each user has 3 profile photos named as follows:
- `{userId}-1.jpg` - Primary profile photo
- `{userId}-2.jpg` - Secondary photo
- `{userId}-3.jpg` - Tertiary photo

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 1080x1080 pixels (or similar high-quality resolution)
- Aspect ratio: Square or portrait preferred

## Adding New Images

1. Place your image files in this directory (`/public/images/profiles/`)
2. Follow the naming convention above
3. The images will automatically be referenced in the app

## Fallback

The app is currently configured to show placeholder images if local images are not found. Once you add your images following the naming convention, they will automatically display.
